05 / FAQ

FAQ

Is this a cloud agent?

No. It's a peer-to-peer sync utility for local Claude Code instances.

What about privacy?

Default, nothing is shared. /connect opts one session in, to specific people. /disconnect — or just ending the session — wipes the shared copy immediately.

Any infrastructure to run?

None. No AWS, no buckets, no servers. Sessions stay on your laptops; we run the thin sync layer in between.

Is there an audit log?

On the roadmap for the Team/Enterprise tiers.
